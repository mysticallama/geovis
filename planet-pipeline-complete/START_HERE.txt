## ğŸ¯ Your First Query

Create a file `my_first_query.py`:

```python
from planet_pipeline import PlanetPipeline

# Initialize (API key from .env automatically)
pipeline = PlanetPipeline(storage_dir="./my_data")

# Add your area of interest
pipeline.add_aoi(
    name="my_site",
    geometry={
        "type": "Polygon",
        "coordinates": [[
            [-122.5, 37.7],  # Replace with your coordinates
            [-122.5, 37.8],
            [-122.4, 37.8],
            [-122.4, 37.7],
            [-122.5, 37.7]
        ]]
    }
)

# Query imagery
results = pipeline.query_all_aois(
    start_date="2024-01-01",  # Replace with your dates
    end_date="2024-01-31"
)

print(f"Found {len(results['my_site'])} scenes!")

# Download (limit to 5 for testing)
pipeline.download_imagery(limit_per_aoi=5)

# Calculate NDVI
pipeline.calculate_indices(indices=["ndvi"])

print("âœ… Complete! Check ./my_data/ for outputs")
```

Run it:
```bash
python my_first_query.py
```

## ğŸ†˜ Troubleshooting

### "Cannot import planet_pipeline"
Make sure you:
1. Ran `SETUP_INSTRUCTIONS.py`
2. Activated your virtual environment (see `(venv)` in terminal)
3. Installed requirements: `pip install -r requirements.txt`

### "Planet API key is required"
Make sure you:
1. Created `.env` file from `.env.example`
2. Added your actual API key (get from https://www.planet.com/account/#/)
3. API key is in format: `PL_API_KEY=your_key_here` (no quotes)

### "pip: command not found"
Use: `python -m pip install -r requirements.txt`

### "requirements.txt not found"
Make sure you're in the `planet-pipeline-complete` folder:
```bash
pwd  # Check current directory
cd path/to/planet-pipeline-complete  # Navigate if needed
```
